# Official Anthropic Skills

Official skills forked from Anthropic. Synced with upstream regularly.

| Skill | Description |
|-------|-------------|
| [algorithmic-art](./algorithmic-art/) | Create algorithmic art using p5.js with seeded randomness, flow fields, and particle systems |
| [brand-guidelines](./brand-guidelines/) | Apply Anthropic's official brand colors and typography to artifacts |
| [canvas-design](./canvas-design/) | Create beautiful visual art in .png and .pdf documents (posters, designs, static pieces) |
| [claude-api](./claude-api/) | Build apps with the Claude API or Anthropic SDK across multiple languages |
| [doc-coauthoring](./doc-coauthoring/) | Structured workflow for co-authoring documentation, proposals, and technical specs |
| [docx](./docx/) | Create, read, edit, and manipulate Word (.docx) documents |
| [frontend-design](./frontend-design/) | Create distinctive, production-grade frontend interfaces (websites, dashboards, React components) |
| [internal-comms](./internal-comms/) | Write internal communications — status reports, newsletters, FAQs, incident reports, etc. |
| [mcp-builder](./mcp-builder/) | Build high-quality MCP (Model Context Protocol) servers in Python or TypeScript |
| [pdf](./pdf/) | Read, merge, split, create, fill, encrypt, and OCR PDF files |
| [pptx](./pptx/) | Create, edit, parse, or convert PowerPoint (.pptx) presentations |
| [skill-creator](./skill-creator/) | Create, improve, evaluate, and benchmark agent skills |
| [slack-gif-creator](./slack-gif-creator/) | Create animated GIFs optimized for Slack |
| [theme-factory](./theme-factory/) | Style artifacts with pre-set or custom themes (colors, fonts) |
| [web-artifacts-builder](./web-artifacts-builder/) | Build elaborate multi-component HTML artifacts using React, Tailwind CSS, and shadcn/ui |
| [webapp-testing](./webapp-testing/) | Test local web applications using Playwright (screenshots, browser logs, UI verification) |
| [xlsx](./xlsx/) | Create, read, edit, and convert spreadsheet files (.xlsx, .xlsm, .csv, .tsv) |
